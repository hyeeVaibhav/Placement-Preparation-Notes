1. Reverse a Number
Question
Write a program to reverse a given number.
Algorithm
  1. Read the number from the user.
  2. Initialize rev = 0.
  3. Extract the last digit using num % 10.
  4. Add the digit to rev → rev = rev * 10 + digit.
  5. Remove the last digit → num = num / 10.
  6. Repeat steps 3 to 5 until num becomes 0.
  7. Print rev.
Code
import java.util.*;
public class ReverseNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter any number : ");
        int num = sc.nextInt();
        int rev = 0, r;
        while (num != 0) {
            r = num % 10;
            rev = rev * 10 + r;
            num = num / 10;
        }
        System.out.println("Reverse : " + rev);
    }
}
Input  12345
Output 54321
Time: O(d) where d = number of digitsSpace: O(1)
Note
  • Any number obtained by reading the digits from right to left.
  • Works for positive integers; handle sign separately for negative numbers.
2. Palindrome Number
Question
Write a program to check whether a number is a palindrome or not.
Algorithm
  1. Read the number from the user.
  2. Store the number in another variable (originalNumber).
  3. Reverse the number (same logic as Program 1).
  4. Compare originalNumber and the reversed value.
  5. If equal → palindrome, else → not palindrome.
Code
import java.util.*;
public class PalindromeNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter any number : ");
        int num = sc.nextInt();
        int original = num;
        int rev = 0, r;
        while (num != 0) {
            r = num % 10;
            rev = rev * 10 + r;
            num = num / 10;
        }
        if (original == rev)
            System.out.println("Number is Palindrome");
        else
            System.out.println("Number is not Palindrome");
    }
}
Input  121
Output Number is Palindrome
Time: O(d) where d = number of digitsSpace: O(1)
Note
  • A number that reads the same forwards and backwards is a palindrome.
  • Examples: 121, 12321, 7, 1001.
3. Prime Number
Question
Write a program to check whether a number is prime or not.
Algorithm
  1. Read the number from the user.
  2. If number ≤ 1 → not prime.
  3. Check divisibility from 2 up to sqrt(n).
  4. If the number is divisible by any i, then it is not prime.
  5. If not divisible by any i → prime.
Code
import java.util.*;
public class PrimeNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter any number : ");
        int n = sc.nextInt();
        boolean isPrime = true;
        if (n <= 1) {
            isPrime = false;
        } else {
            for (int i = 2; i * i <= n; i++) {
                if (n % i == 0) {
                    isPrime = false;
                    break;
                }
            }
        }
        if (isPrime)
            System.out.println(n + " is a Prime");
        else
            System.out.println(n + " is not a Prime");
    }
}
Input  17
Output 17 is a Prime
Time: O(√n)Space: O(1)
Note
  • A prime number has exactly two factors: 1 and itself.
  • It is enough to test divisibility up to √n.
  • Examples: 2, 3, 5, 7, 11, 13, 17.
4. Armstrong Number
Question
Write a program to check whether a number is an Armstrong number or not.
Algorithm
  1. Read the number from the user.
  2. Count the total number of digits in the number.
  3. For each digit, calculate (digit ^ count) and add it to sum.
  4. If sum == original number → Armstrong, else → not Armstrong.
Code
import java.util.*;
public class ArmstrongNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter any number : ");
        int n = sc.nextInt();
        int temp = n, sum = 0, count = 0;
        while (temp > 0) {
            temp = temp / 10;
            count++;
        }
        temp = n;
        while (temp > 0) {
            int digit = temp % 10;
            sum = sum + (int) Math.pow(digit, count);
            temp = temp / 10;
        }
        if (sum == n)
            System.out.println(n + " is Armstrong number");
        else
            System.out.println(n + " is not Armstrong number");
    }
}
Input  153
Output 153 is Armstrong number
Time: O(d) where d = number of digitsSpace: O(1)
Note
  • An Armstrong number equals the sum of its own digits, each raised to the power of the total digit count.
  • Examples: 0, 1, 153, 370, 371, 407.

7. Fibonacci Series
Question
Write a program to print the Fibonacci series up to n terms.
Algorithm
  1. Read the number of terms n from the user.
  2. Initialize first = 0, second = 1.
  3. Print the first two terms.
  4. For i = 3 to n: next = first + second, print next, first = second, second = next.
  5. Repeat step 4 until i > n.
Code
import java.util.*;
public class FibonacciSeries {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of terms : ");
        int n = sc.nextInt();
        int first = 0, second = 1, next;
        System.out.print(first + " " + second + " ");
        for (int i = 3; i <= n; i++) {
            next = first + second;
            System.out.print(next + " ");
            first = second;
            second = next;
        }
    }
}
Input  7
Output 0 1 1 2 3 5 8
Time: O(n)Space: O(1)
Note
  • The Fibonacci series starts with 0 and 1.
  • Every following term is the sum of the previous two terms.

14. Check Two Strings are Anagrams
Question
Write a program to check whether two strings are anagrams of each other.
Algorithm
  1. Read two strings from the user.
  2. If their lengths are different → not an anagram.
  3. Sort the characters of both strings.
  4. If the sorted strings are equal → anagram.
  5. Else → not an anagram.
Code
import java.util.*;
public class AnagramString {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter first string : ");
        String str1 = sc.nextLine();
        System.out.print("Enter second string : ");
        String str2 = sc.nextLine();
        if (str1.length() != str2.length()) {
            System.out.println("Not Anagrams");
            return;
        }
        char[] ch1 = str1.toCharArray();
        char[] ch2 = str2.toCharArray();
        Arrays.sort(ch1);
        Arrays.sort(ch2);
        if (Arrays.equals(ch1, ch2))
            System.out.println("Strings are Anagrams");
        else
            System.out.println("Strings are not Anagrams");
    }
}
Input  listen silent
Output Strings are Anagrams
Time: O(n log n)Space: O(n)
Note
  • Two strings can only be anagrams if they have equal length.
  • Sorting both strings and comparing them is the most common approach.

Algorithm
  1. Read two numbers a and b.
  2. Find the greater number.
  3. Initialize lcm = greater.
  4. While (true): if (lcm % a == 0 && lcm % b == 0) break; else lcm++.
  5. Print lcm.
Code
import java.util.*;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter first number : ");
        int a = sc.nextInt();
        System.out.print("Enter second number : ");
        int b = sc.nextInt();
        int greater = (a > b) ? a : b;
        int lcm = greater;
        while (true) {
            if (lcm % a == 0 && lcm % b == 0)
                break;
            lcm++;
        }
        System.out.println("LCM is : " + lcm);
    }
}

Algorithm
  1. Read two numbers a and b.
  2. While (b != 0): rem = a % b, a = b, b = rem.

Code
import java.util.*;

            int rem = a % b;
            a = b;
            b = rem;

    }
}

19. Bubble Sort
Question
Write a program to sort an array of numbers using Bubble Sort.
Algorithm
  1. Read the array from the user.
  2. For i = 0 to n-2, for j = 0 to n-i-2:
  3. If arr[j] > arr[j+1], swap arr[j] and arr[j+1].
  4. After each outer pass the largest remaining element settles at the end.
  5. Print the sorted array.
Code
import java.util.*;
public class BubbleSort {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of elements : ");
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) arr[i] = sc.nextInt();
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
        System.out.println("Sorted array : " + Arrays.toString(arr));
    }
}
Input  5\n5 1 4 2 8
Output [1, 2, 4, 5, 8]
Time: O(n²)Space: O(1)
Note
  • Bubble Sort repeatedly swaps adjacent out-of-order elements.
  • Simple to explain, but not efficient for large arrays.
20. Binary Search
Question
Write a program to search an element in a sorted array using Binary Search.
Algorithm
  1. Read a sorted array and the key to search.
  2. Set low = 0 and high = n - 1.
  3. While low ≤ high: find mid = (low + high) / 2.
  4. If arr[mid] == key, element found.
  5. If arr[mid] < key, search the right half, else search the left half.
Code
import java.util.*;
public class BinarySearch {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of elements : ");
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) arr[i] = sc.nextInt();
        System.out.print("Enter element to search : ");
        int key = sc.nextInt();
        int low = 0, high = n - 1, result = -1;
        while (low <= high) {
            int mid = (low + high) / 2;
            if (arr[mid] == key) { result = mid; break; }
            else if (arr[mid] < key) low = mid + 1;
            else high = mid - 1;
        }
        if (result != -1)
            System.out.println("Element found at index : " + result);
        else
            System.out.println("Element not found");
    }
}
Input  6\n2 4 6 8 10 12, key = 10
Output Element found at index : 4
Time: O(log n)Space: O(1)
Note
  • Binary Search only works on a sorted array.
  • It halves the search space on every comparison, which is why it is much faster than a linear scan.
21. Second Largest Number in an Array
Question
Write a program to find the second largest number in an array.
Algorithm
  1. Read the array from the user.
  2. Initialize largest and secondLargest to the smallest possible value.
  3. Traverse the array; if element > largest, update secondLargest = largest, largest = element.
  4. Else if element > secondLargest and element != largest, update secondLargest.
  5. Print secondLargest.
Code
import java.util.*;
public class SecondLargest {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter number of elements : ");
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) arr[i] = sc.nextInt();
        int largest = Integer.MIN_VALUE, second = Integer.MIN_VALUE;
        for (int num : arr) {
            if (num > largest) {
                second = largest;
                largest = num;
            } else if (num > second && num != largest) {
                second = num;
            }
        }
        System.out.println("Second largest number is : " + second);
    }
}
Input  5\n12 35 1 10 34
Output Second largest number is : 34
Time: O(n)Space: O(1)
Note
  • The array is traversed only once, tracking both the largest and second largest values together instead of sorting the whole array.

Algorithm
  1. Read the number from the user.
  2. Initialize sum = 0.
  3. For i = 1 to n/2, if n % i == 0, add i to sum.

Code
import java.util.*;

                sum = sum + i;
            }
        }
        if (sum == n)
            System.out.println(n + " is a Perfect number");
        else
            System.out.println(n + " is not a Perfect number");
    }
}
Input  28
Output 28 is a Perfect number
Time: O(n)Space: O(1)
Note
  • A Perfect number equals the sum of its proper divisors (excluding itself).
  • Examples: 6, 28, 496, 8128.

Algorithm
  1. Read the dimensions and elements of matrix A and matrix B.
  2. Check that the number of columns of A equals the number of rows of B.
  3. For every result cell [i][j], sum the products of the matching row of A and column of B.
  4. Store the sum in the result matrix.
  5. Print the result matrix.
Code
import java.util.*;
public class MatrixMultiplication {
    public static void main(String[] args) {
        int[][] a = {{1, 2}, {3, 4}};
        int[][] b = {{5, 6}, {7, 8}};
        int rowsA = a.length, colsA = a[0].length, colsB = b[0].length;
        int[][] result = new int[rowsA][colsB];
        for (int i = 0; i < rowsA; i++) {
            for (int j = 0; j < colsB; j++) {
                for (int k = 0; k < colsA; k++) {
                    result[i][j] += a[i][k] * b[k][j];
                }
            }
        }
        System.out.println("Resultant Matrix :");
        for (int[] row : result) {
            System.out.println(Arrays.toString(row));
        }
    }
}
Input  A = {{1,2},{3,4}}, B = {{5,6},{7,8}}
Output [19, 22] [43, 50]
Time: O(n³) for n x n matricesSpace: O(n²)
Note
  • Matrix multiplication needs three nested loops.
  • The number of columns in the first matrix must equal the number of rows in the second.

Algorithm
  1. Read the string from the user.
  2. Create a HashMap to store each character and its frequency.
  3. Traverse the string and update the count for every character.
  4. Print only the characters whose count is greater than 1.
Code
import java.util.*;
public class DuplicateCharacters {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter a string : ");
        String str = sc.nextLine();
        HashMap<Character, Integer> map = new HashMap<>();
        for (char ch : str.toCharArray()) {
            map.put(ch, map.getOrDefault(ch, 0) + 1);
        }
        System.out.println("Duplicate characters :");
        for (Map.Entry<Character, Integer> entry : map.entrySet()) {
            if (entry.getValue() > 1) {
                System.out.println(entry.getKey() + " : " + entry.getValue());
            }
        }
    }
}

26. Check Harshad (Niven) Number
Question
Write a program to check whether a number is a Harshad (Niven) number or not.
Algorithm
  1. Read the number from the user.
  2. Calculate the sum of its digits.
  3. Check if the number is exactly divisible by the digit sum.
  4. If divisible → Harshad number, else → not a Harshad number.
Code
import java.util.*;
public class HarshadNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter any number : ");
        int n = sc.nextInt();
        int temp = n, sum = 0;
        while (temp > 0) {
            sum += temp % 10;
            temp /= 10;
        }
        if (n % sum == 0)
            System.out.println(n + " is a Harshad number");
        else
            System.out.println(n + " is not a Harshad number");
    }
}
Input  21
Output 21 is a Harshad number
Time: O(d) where d = number of digitsSpace: O(1)
Note
  • A Harshad (or Niven) number is divisible by the sum of its own digits.
  • For example, 21 is a Harshad number because 21 / (2+1) = 7.
